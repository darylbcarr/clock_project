Overview

What this clock is and how it works

This is an ESP32-S3 powered analog clock with a stepper-motor-driven minute hand, an optical position sensor, an OLED display, and two independently controllable LED strips. It connects to your WiFi network, syncs time automatically over the internet, and is controllable from any browser on your network — as well as from Alexa, Apple Home, or Google Home via the Matter smart-home standard.

🕐
Accurate Timekeeping

Syncs to internet time servers (SNTP) automatically. The optical sensor corrects minor drift near the top of each hour — no manual adjustments needed.

💡
Dual LED Strips

Independent Ring and Base strips with 8 animated effects, full colour control, and brightness adjustment. Works with Alexa and other smart-home platforms.

🌐
Web Interface

A full-featured single-page app accessible from any browser on your network. No app install required. Updates live every second.

🏠
Smart Home Ready

Matter-certified. Commission once with Alexa, Apple Home, or Google Home and control the lights with voice commands or automations.

First-Time Setup

Complete walkthrough from power-on to a fully working clock

The setup sequence below is the recommended order for a freshly flashed device. Each step builds on the previous one. Most steps only need to be done once.

✓

Once setup is complete, the clock maintains accurate time automatically via internet sync — no manual time-setting is needed after power cycles.

Step 1 — Connect to WiFi

On first boot the display shows a setup screen. Choose a connection method and follow the prompts. See the Network Connection section for full details.

Step 2 — Find the Device Address

Once WiFi is connected, navigate on the clock display:

📟
On the clock OLED
Main Menu → Status → Network
Shows the IP address and mDNS hostname (e.g. clock_a1b2)

Open a browser and navigate to http://192.168.x.x or http://clock_a1b2.local.

Step 3 — Check for Firmware Updates

  1. 1
    Go to the Config tab in the web interface.
  2. 2
    Scroll to the Firmware Update card.
  3. 3
    Press Check for Updates. If a newer version is shown, press Update Now.
  4. 4
    Watch the OLED display for progress. The device reboots automatically when done.
⚠

Do not power off the clock during an update. Wait for the OLED to show the completion message.

Step 4 — Set a Hostname (Optional)

On the Config tab, find Device Hostname. Replace the default (e.g. clock_a1b2) with something memorable like office_clock. Press Save — the new address takes effect immediately at http://office_clock.local.

💡

Use lowercase letters, numbers, and underscores or hyphens only. Avoid spaces and special characters.

Step 5 — Verify Motor Direction

  1. 1
    On the Clock tab, find the Fine Tune card and press 1 Min Fwd →.
  2. 2
    Watch the minute hand. It should move clockwise.
  3. 3
    If it moved counter-clockwise, go to Config → Motor Direction, select Reverse, and press Save.
  4. 4
    Test again with 1 Min Fwd → to confirm.

Step 6 — Calibrate the Optical Sensor

The sensor detects a reflective slot on the clock ring to know where 12:00 is. Calibration is two steps.

ⓘ

Perform calibration under the normal ambient lighting the clock will operate in. Avoid shining a direct light at the sensor.

Step A — Background Sample
  1. 1
    Ensure the minute hand is not over the sensor slot.
  2. 2
    Press Sample Background.
  3. 3
    The ring advances ~10 minutes while sampling (~12 seconds).
  4. 4
    Result shows Mean and Threshold values.
Step B — Find Slot & Set Offset
  1. 1
    Press Find Slot ▶. The ring advances until the slot is detected.
  2. 2
    Use the ◀ Back and Fwd ▶ buttons to position the hand exactly at 12:00.
  3. 3
    Press Save Offset when the hand is correctly positioned.

Step 7 — Set the Clock to the Correct Time

  1. 1
    On the Clock tab, find the Observed Clock Position card.
  2. 2
    Look at the physical clock face and note where the minute hand is pointing.
  3. 3
    Enter that time in the input field and press Set Clock to Actual Time.
  4. 4
    The motor advances the hands to the correct current time.
✓

Setup complete. Once the green ● Internet synced badge appears on the Clock tab, the clock maintains accurate time automatically. The sensor performs an automatic drift correction near the top of each hour.

Option 1

Setup WiFi

Enter your SSID and password directly on the device using the rotary encoder and buttons. Simplest option — works without any smart-home app.

Option 2

Matter

Commission via Amazon Alexa, Apple Home, or Google Home over Bluetooth. Integrates the clock with your smart-home platform from the start.

Finding the Device

How to locate and open the web interface

After connecting to WiFi, navigate on the clock display to find its address:

📟
Main Menu → Status → Network

The screen shows both the IP address and the mDNS hostname. Either can be used in a browser:

http://192.168.50.42
http://clock_a1b2.local
💡

.local addresses: Work natively on macOS, iOS, Android, and most Linux systems. On Windows they require Bonjour (bundled with iTunes) or Windows 11's built-in mDNS support.

The web interface updates automatically every second — there is no need to refresh the page.

Lights Tab

Control the Ring and Base LED strips — colour, brightness, and effects

The clock has two independent WS2812B LED strips:

Ring

The LED ring around the clock face (default 24 LEDs)

Base

The LEDs underneath the clock body (default 6 LEDs)

Use the APPLY TO selector at the top of the Lights tab to target Ring, Base, or Both strips at once. All colour, brightness, and effect changes apply to the selected target.

Colour

A palette of 21 colour presets is shown as swatches. Click a swatch to apply that colour immediately. The colours are calibrated to match the named colours that Alexa sends via Matter — what you set here is exactly what you'll get when you ask Alexa for the same colour.

Brightness

A slider from 0 to 255. Changes take effect as you drag. The setting is saved automatically.

Device Hostname

The mDNS name used to reach the clock on the local network. Default is derived from the MAC address (e.g. clock_a1b2). Change to something memorable and press Save. Takes effect immediately — no restart required.

Update the stored WiFi network name and password without using the physical buttons. Enter the new SSID and password and press Save & Restart. The connection will drop briefly during restart.

Sensor Tab

Calibrate the optical position sensor for accurate hourly drift correction

The optical sensor uses an LED and a light-dependent resistor (LDR) to detect a reflective slot on the clock ring as it passes 12:00 each hour. The clock uses this to automatically correct up to a few minutes of drift without any user intervention.

ⓘ

Calibration only needs to be done once during initial setup. Redo it if the clock moves to a significantly different lighting environment, or if you notice the hands jumping at the top of the hour.

Sample Background

Measures ambient light across multiple ring positions to establish the detection threshold.

  1. 1
    Ensure the minute hand is not directly over the sensor slot (roughly anywhere except 12:00).
  2. 2
    Press Sample Background.
  3. 3
    The ring advances approximately 10 minutes while sampling (~12 seconds).
  4. 4
    The result shows the measured Mean and calculated Threshold.

Find Slot / Set Offset

Locates the reflective slot and calibrates its exact position relative to 12:00.

  1. 1
    Press Find Slot ▶. The ring advances until the slot is detected.
  2. 2
    Use ◀ Back and Fwd ▶ to position the minute hand exactly at 12:00 on the clock face.
  3. 3
    Press Save Offset when correctly positioned.
💡

If the slot is not found, check sensor wiring and ensure the background was sampled correctly first. Press Try Again to retry.

Read Now

Takes a single sensor reading and displays the raw ADC value. Use this to verify the sensor detects the slot when the ring is manually positioned over it.

Live Scan

Advances the ring a set number of minutes and plots the ADC reading at each step. Useful for diagnosing sensor issues or confirming the slot is clearly visible above the background threshold.

Alexa, Apple Home, and Google Home control

The clock appears on your smart-home network as two Extended Color Light devices:

Clock Ring

Controls the ring LED strip around the clock face

Clock Base

Controls the base LED strip underneath the clock

What You Can Control via Matter

ControlBehaviour
On / OffTurns the strip on or off
BrightnessSets light level (0–100%)

Colour TemperatureSets warm to cool white tones

Effects and Matter

ⓘ

Matter commands always use Static mode. When Alexa or another controller sends a colour or brightness command, the strip switches to a solid, static colour — any running effect (Breathe, Rainbow, etc.) is replaced. This is by design: smart-home platforms have no concept of animated effects.

Practical workflow for using both:

  1. 1
    Use the web interface Lights tab to choose an effect (e.g. Breathe in warm white). Effects are saved and resume after power cycles.
  2. 2
    Use Alexa / Matter to turn the strip on and off, or to switch to a specific solid colour for a scene.
  3. 3
    If you want the effect back after a Matter command, return to the web interface and re-select it.

Colour Accuracy

Alexa sends colours using the XY colour space. The clock converts these to RGB and normalises to full brightness, then applies the configured brightness separately. The colour swatches on the Lights tab are calibrated to match Alexa's named colours exactly.

Adding the Clock to a Second Smart-Home Platform

  1. 1
    Navigate to the Info tab and find the Matter section.
  2. 2
    Press Open Commissioning Window.
  3. 3
    Scan the QR code or enter the pairing code in the second platform's app.

Changing WiFi Credentials

What to do when you replace your router or change your network name / password

There are three ways to update the stored WiFi credentials:

Method 1 — Web Interface (easiest)

While the clock is still connected to the old network:

  1. 1
    Go to Config → WiFi Credentials.
  2. 2
    Enter the new SSID and password.
  3. 3
    Press Save & Restart. The clock restarts and connects to the new network.
Method 2 — On-device menu

Navigate to Main Menu → Network → Set WiFi and enter new credentials using the encoder text-entry interface.

Method 3 — WiFi not found screen

If the clock fails to connect at boot (wrong password, new router), a recovery screen appears:

WiFi not found!

Factory Reset

Clear all network and commissioning data

A factory reset clears WiFi credentials, Matter fabric data, and commissioning flags. The device restarts into first-time setup.

✓

LED effects, motor speed, motor direction, sensor calibration, and hostname are preserved after a factory reset. Only network and commissioning data is erased.

How to Perform a Factory Reset

InputDurationResult
Hold A + B 5 seconds Display shows WiFi Reset! / Restarting… then device restarts into first-time setup
Hold encoder button 5 seconds Same as above
⚠

After a factory reset, the clock will need to be re-commissioned in any smart-home app (Alexa, Apple Home, Google Home) it was previously paired with.

Troubleshooting

Common issues and how to resolve them

Clock shows wrong time

Check the ● Internet synced badge on the Clock tab. If it is not shown, the clock has not received a time sync — verify WiFi is connected on the Info tab. If WiFi is connected but the time is wrong, check the Timezone field; set an override in Config if geolocation is incorrect.

If the physical hands are misaligned with the displayed time, use the Observed Clock Position card on the Clock tab to re-align them.

Minute hand moves the wrong direction

Go to Config → Motor Direction, select Reverse, and press Save.

Hands jump backward at the top of the hour

The optical sensor is triggering outside the expected window. Re-run the Sample Background step in the Sensor tab under normal ambient lighting conditions, then redo the Find Slot / Set Offset calibration.

Can't reach the web interface
  • Confirm the clock is connected to WiFi via Main Menu → Status → Network.
  • Try the IP address instead of the .local hostname.
  • On Windows, .local addresses require Bonjour or Windows 11.
  • Ensure your phone or computer is on the same WiFi network as the clock.
LEDs not working or showing wrong colours

Verify the strip lengths in Config → LED Strip Lengths match your actual hardware. If Alexa colours look different from the web swatches, they should match — the swatches are calibrated to Alexa's XY colour values.

If a Matter command recently changed the colour, the effect was replaced with Static. Re-select the desired effect on the Lights tab.

WiFi not connecting after router change

See the Changing WiFi Credentials section. The quickest method is the web interface Config tab while the clock is still reachable on the old network. If the network is already gone, use the WiFi not found recovery screen at boot.

Clock not moving after power cycle

The clock waits for an internet time sync (SNTP) before starting the motor. This prevents the hands from advancing from an incorrect starting position. Once WiFi connects and the time syncs, the motor will start. If WiFi is unavailable, the clock will not run — connect it to a network for full functionality.
