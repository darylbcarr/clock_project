Overview

What this clock is and how it works

This is an ESP32-S3 powered analog clock with a stepper-motor-driven minute hand, an optical position sensor, an OLED display, and two independently controllable LED strips. It connects to your WiFi network, syncs time automatically over the internet, and is controllable from any browser on your network — as well as from Alexa, Apple Home, or Google Home via the Matter smart-home standard.

🕐
Accurate Timekeeping

Syncs to internet time servers automatically. The optical sensor corrects minor drift near the top of each hour — no manual adjustments needed.

💡
Dual LED Strips

Independent Ring and Base strips with 7 animated effects, full colour control, and brightness adjustment. Works with Alexa and other smart-home platforms.

🌐
Web Interface

A full-featured single-page app accessible from any browser on your network. No app install required. Updates live every second via WebSocket.

🏠
Smart Home Ready

Matter-certified. Commission once with Alexa, Apple Home, or Google Home and control the lights with voice commands or automations.

First-Time Setup

Complete walkthrough from power-on to a fully working clock

The setup sequence below is the recommended order for a freshly flashed device. Each step builds on the previous one. Most steps only need to be done once.

✓

Once setup is complete, the clock maintains accurate time automatically via internet sync — no manual time-setting is needed after power cycles.

Step 1 — Connect to WiFi

On first boot the display shows a setup screen. Choose a connection method and follow the prompts. See Network Connection for full details.

Step 2 — Find the Device Address

Once WiFi is connected, navigate on the clock display:

📟
On the clock OLED
Main Menu → Status → Network
Shows the IP address and mDNS hostname (e.g. clock_a1b2)

Open a browser and navigate to http://192.168.x.x or http://clock_a1b2.local.

Step 3 — Check for Firmware Updates

  1. 1
    Go to the Config tab in the web interface.
  2. 2
    Scroll to the Firmware Update card.
  3. 3
    Press Check for Updates. If a newer version is available, press Update Now.
  4. 4
    Watch the OLED display for progress. The device reboots automatically when done.
⚠

Do not power off the clock during an update. Wait for the OLED to show the completion message.

Step 4 — Set a Hostname (Optional)

On the Config tab, find Device Hostname. Replace the default (e.g. clock_a1b2) with something memorable like office_clock. Press Save — the new address takes effect immediately at http://office_clock.local.

💡

Use lowercase letters, numbers, and hyphens only. Avoid spaces and special characters.

Step 5 — Verify Motor Direction

  1. 1
    On the Clock tab, find the Fine Tune card and press 1 Min Fwd.
  2. 2
    Watch the minute hand. It should move clockwise.
  3. 3
    If it moved counter-clockwise, go to Config → Motor Direction, select Reverse.
  4. 4
    Test again with 1 Min Fwd to confirm.

Step 6 — Calibrate the Optical Sensor

Found in Config → Sensor Calibration. The sensor detects a reflective slot on the clock ring to know where 12:00 is. Calibration is two steps.

ⓘ

Perform calibration under the normal ambient lighting the clock will operate in. Avoid shining a direct light at the sensor.

Step A — Background Sample
  1. 1
    Ensure the minute hand is not over the sensor slot (not at 12:00).
  2. 2
    Press Sample Background.
  3. 3
    The ring advances ~10 minutes while sampling (~12 seconds).
  4. 4
    Result shows Mean and Threshold values.
Step B — Find Slot & Set Offset
  1. 1
    Press Find Slot ▶. The ring advances until the slot is detected.
  2. 2
    Use ◀ Back and Fwd ▶ to position the hand exactly at 12:00.
  3. 3
    Press Save Offset when correctly positioned.

Step 7 — Set the Clock to the Correct Time

  1. 1
    On the Clock tab, find the Observed Clock Position card.
  2. 2
    Look at the physical clock face and note where the minute hand is pointing.
  3. 3
    Enter that time in the input field and press Set Clock to Actual Time.
  4. 4
    The motor advances the hands to the correct current time.
✓

Setup complete. Once the green ● Internet synced badge appears on the Clock tab, the clock maintains accurate time automatically. The sensor performs automatic drift correction near the top of each hour.

Config Tab

Persistent settings — all changes survive power cycles

LED Strip Lengths

Sets the active LED count for each strip. Enter the number and press Apply. This affects how many LEDs the effects use — set it to match your actual hardware.

Motor Direction

Select Normal if the minute hand advances clockwise, or Reverse if it goes counter-clockwise. The change takes effect immediately.

Sensor Calibration

The optical sensor uses an LED and LDR to detect a reflective slot on the clock ring as it passes 12:00 each hour. Calibration sets the detection threshold and the exact 12:00 offset. Only needs to be done once — redo it if the clock moves to a significantly different lighting environment.

ⓘ

Calibrate under the normal ambient lighting the clock will operate in. Avoid shining a direct light at the sensor during sampling.

Step 1 — Background Sample

Establishes the detection threshold by sampling the sensor across multiple ring positions.

  1. 1
    Ensure the minute hand is not directly over the sensor slot (not at 12:00).
  2. 2
    Press Sample Background.
  3. 3
    The ring advances approximately 10 minutes while sampling (~12 seconds).
  4. 4
    The result shows the measured Mean and calculated Threshold.
Step 2 — Slot Offset

Locates the reflective slot and calibrates its exact position relative to 12:00.

  1. 1
    Press Find Slot ▶. The ring advances until the slot triggers the sensor.
  2. 2
    Use ◀ Back and Fwd ▶ to step the minute hand to exactly 12:00 on the clock face.
  3. 3
    Press Save Offset. The saved offset (in steps) is shown below the buttons.
💡

If the slot is not found, check sensor wiring and ensure the background was sampled first. Press Try Again to retry.

Motor Speed

Controls how fast the stepper motor turns, in microseconds per half-step. Drag the slider and press Save Speed.

ValueEffect
900 µs (min)Fastest rotation — noisiest
2000 µs (default)~8 seconds per revolution — balanced
3000 µs (max)Slowest, quietest — recommended for bedroom use
💡

The slider will not be overwritten while you are dragging it, even though the page receives live updates every second.

Timezone Override

Normally the clock detects its timezone automatically via IP geolocation. To force a specific timezone, enter a POSIX TZ string and press Save. Leave blank to use automatic detection.

Device Hostname

The mDNS name used to reach the clock on the local network (e.g. clock_a1b2.local). Change to something memorable and press Save. Takes effect immediately — no restart required.

WiFi Credentials

Update the stored WiFi network name and password without using the physical buttons. Enter the new SSID and password and press Save & Restart. The connection will drop briefly during restart. Matter commissioning data is preserved.

Logs Tab

In-memory event log with category filters and auto-refresh

The Logs tab shows a live view of the device's internal event ring buffer. It is useful for diagnosing sensor behaviour, time-sync issues, and light control activity.

ⓘ

The buffer holds the 200 most recent entries and is stored in flash — entries survive reboots. Entries are shown newest first. The buffer wraps when full, discarding the oldest entries.

Category Filter Buttons

Click a category pill to toggle it on or off. Only entries in enabled categories are shown. Filter settings are saved to the device and persist across reboots.

Effects and Matter

ⓘ

Matter commands always use Static mode. When Alexa or another controller sends a colour or brightness command, the strip switches to a solid, static colour — any running effect (Breathe, Rainbow, etc.) is replaced. Smart-home platforms have no concept of animated effects.

Practical workflow for using both:

  1. 1
    Use the Lights tab to choose an effect. Effects are saved and resume after power cycles.
  2. 2
    Use Alexa / Matter to turn the strip on and off, or switch to a specific solid colour for a scene.
  3. 3
    If you want the effect back after a Matter command, return to the Lights tab and re-select it.

Colour Accuracy

Alexa sends colours using the XY colour space. The clock converts these to RGB and normalises to full brightness, then applies the configured brightness separately. The colour swatches on the Lights tab are calibrated to match Alexa's named colours exactly.

A factory reset clears WiFi credentials, Matter fabric data, and commissioning flags. The device restarts into first-time setup.

Troubleshooting

Common issues and how to resolve them

Clock shows wrong time

Check the ● Internet synced badge on the Clock tab. If absent, the clock has not received a time sync — verify WiFi is connected on the Info tab. If WiFi is connected but the time is wrong, check the Timezone card on the Clock tab; set an override in Config if geolocation is incorrect.

If the physical hands are misaligned with the displayed time, use Observed Clock Position on the Clock tab to re-align them.

Minute hand moves the wrong direction

Go to Config → Motor Direction and select Reverse.

Hands jump backward at the top of the hour

The optical sensor is triggering outside the expected window. Re-run the Sample Background step in Config → Sensor Calibration under normal ambient lighting, then redo Find Slot / Set Offset.

Can't reach the web interface
  • Confirm the clock is connected to WiFi via Main Menu → Status → Network.
  • Try the IP address instead of the .local hostname.
  • On Windows, .local addresses require Bonjour or Windows 11.
  • Ensure your phone or computer is on the same WiFi network as the clock.
LEDs not working or showing wrong colours

Verify the strip lengths in Config → LED Strip Lengths match your actual hardware.

If a Matter command recently changed the colour, the effect was replaced with Static. Re-select the desired effect on the Lights tab.

WiFi not connecting after router change

See Changing WiFi Credentials. The quickest method is the web interface Config tab while the clock is still reachable on the old network. If the network is already gone, use the WiFi not found recovery screen at boot.

Clock not moving after power cycle

The clock waits for an internet time sync (SNTP) before starting the motor. This prevents advancing from an incorrect position. Once WiFi connects and time syncs, the motor starts. If WiFi is unavailable, the clock will not run.
