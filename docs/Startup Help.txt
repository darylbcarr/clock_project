  Get IP or DNS address
Go to browser
  connect with IP or DNS name

  Check for Updates
Web->Config->DEVICE HOSTNAME
  Change name as desired, reload web
Web->Config->Update Now
  Watch display screen for validation
Web->Clock
  Press 1 Min Fwd and watch minute hand
  If hand goes forward then done.
  If hand goes backward then...
  Web->Config->MOTOR DIRECTION
    Select Reverse, check by moving as above.
Web->Config->SENSOR CALIBRATION
  Sample Background, hand will move about 10 minutes and show Mean: xxx Threshold: xxx
Web->Config->SENSOR CALIBRATION
  Find Slot-> The hand should move close to 12 then
  Move FWD continuously until exactly at 12, then Save Offset
